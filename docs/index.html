AgentValue-AI

AI-powered toolkit for employee value assessment and talent analytics.

Python Vue 3 FastAPI LangGraph

AI Evaluation Agent

LangGraph-based agent producing three separated views: employee, manager, and audit.

Human-in-the-Loop Approval

All AI outputs require manager approval before taking effect; high-risk items go through HR review.

Multimodal Input

Accepts daily reports, task progress, code contributions, meeting notes, screenshots, and voice.

Observable & Auditable

Prometheus metrics, Grafana dashboards, Langfuse tracing, and append-only audit logs.
