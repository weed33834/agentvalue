# 第三方代码与致谢

本项目在开发过程中参考了以下开源项目的设计理念与实现模式:

| 项目 | 许可证 | 参考内容 |
|---|---|---|
| [opencode](https://github.com/sst/opencode) | MIT | SSE 事件流协议、Session ReAct 循环、工具调用架构 |
| [Open WebUI](https://github.com/open-webui/open-webui) | MIT | 对话界面交互设计、消息渲染方案 |
| [LobeChat](https://github.com/lobehub/lobe-chat) | MIT | 会话管理、主题切换设计参考 |
| [Dify](https://github.com/langgenius/dify) | Apache 2.0 | Provider 管理、知识库、工作流编排架构 |
| [Langfuse](https://github.com/langfuse/langfuse) | MIT | 链路追踪、Token 用量统计设计 |

所有参考仅涉及设计模式与架构思路,代码均为本项目独立实现,未包含任何第三方源代码副本。
